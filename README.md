# 📘 Introduction to Programming and Scripting with Python

Welcome to the course repository! This repository contains core curriculum details, learning objectives, and the final capstone project.

## 📝 Course Description
This course introduces the fundamental principles of computer programming and software development using Python. Designed for beginners, the curriculum covers essential core concepts, including data types, variables, decision-making statements, loops, functions, and file handling. Through practical, hands-on scripting exercises, students learn how to leverage common scripting language constructs—such as lists, literals, and regular expressions—to process data, automate tasks, and build functional applications.

## 🎯 Key Learning Objectives
* **Foundational Syntax:** Master variables, conditional logic, and control flow loops.
* **Modular Code:** Write clean, reusable programs utilizing functions.
* **Data Persistence:** Read, write, and manipulate external data files.
* **Text Processing:** Parse and analyze input using regular expressions and lists.
* **Portfolio Development:** Deploy active software projects using Git and GitHub.

## 🛠️ Technologies & Concepts Covered
* **Language:** Python 3.x
* **Logic:** `if/elif/else`, `while` and `for` loops
* **Structures:** Lists, Literals, Strings, Dictionaries
* **Advanced Utilities:** `re` module (Regular Expressions), Built-in File I/O (`open()`)

## 🎮 Capstone Project: Text-Based Adventure Game
The final milestone of this course is a terminal-based narrative game that synthesizes all course concepts into a single, interactive application. 

The complete source code for this project is located in the **[TextBasedGame](./TextBasedGame)** directory of this repository.

### How to Play the Game

1. **Clone this repository to your local machine:**
   ```bash
   git clone https://github.com
   ```

2. **Navigate into the game directory:**
   ```bash
   cd Python-class/TextBasedGame
   ```

3. **Run the game file:**
   ```bash
   python main.py
   ```
   *(Note: Replace `main.py` with the exact name of your Python script if it is named differently, like `game.py` or `adventure.py`).*

### Code Structure
The game leverages the following programming techniques taught in class:
* **Input Validation:** Uses regular expressions to process user commands cleanly.
* **State Tracking:** Uses Python variables and lists to monitor inventory and locations.
* **File Handling:** Saves or reads external text to manage game files or score logs.
